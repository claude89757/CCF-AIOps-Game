# CCF-AIOps-Game

## 赛道一: 基于大模型智能体的微服务根因定位

本赛题旨在评估大模型在根因定位任务中的能力。参赛者将基于微服务系统数据，构建一个能够理解自然语言查询、融合监控指标、服务日志和调用链数据的大模型故障诊断智系统，自动识别异常背后的根因组件和故障类型。
